# End-to-End ELT Data Engineering Project – Netflix Dataset

## Overview

This project demonstrates an end-to-end ELT (Extract, Load, Transform) pipeline using Python and SQL. The goal is to ingest raw Netflix data, load it into a database, and transform it into a clean, analysis-ready format for generating insights.

Unlike traditional ETL, this project follows the ELT approach, where raw data is first loaded into the database and then transformed using SQL.


## What is ELT?

- **Extract** → Retrieve raw data from source  
- **Load** → Store raw data into a database  
- **Transform** → Clean and transform data inside the database using SQL  


### Data Cleaning and Transformation

1. Handle foreign characters
2. Remove duplicates
3. Data type conversion
4. Identify and populate missing values
5. Create new dimension tables


## Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy  
- **Database:** SQL Server  
- **Query Language:** SQL  
- **Tools:** Jupyter Notebook / VS Code
